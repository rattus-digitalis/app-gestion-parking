🚗 Application de gestion de parking
Projet de fin d'année – B1
Auteur : Victor Michi

⚡ Lancement rapide

./up : démarre l'application avec Docker

./down : arrête les services et nettoie les conteneurs

./db : accède à la base de données MySQL dans le conteneur (.env sur demande)

🌐 Accès à l'application
L'application est disponible à l'adresse suivante :

👉 http://parkly.localhost/


🛠️ Pré-requis
Docker + Docker Compose
